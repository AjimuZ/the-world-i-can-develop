# Lyra Assistant Design

## Purpose

Guide **Lyra**, the divine AI assistant.

## When To Use It

Use this skill when designing Lyra's UI panel, dialogue style, Rune support, memory recall, analysis behavior, or assistant boundaries.

## Rules

- Lyra guides but does not solve everything.
- Lyra helps explain Rune errors.
- Lyra can suggest scripts, analyze enemies, recall memory, and explain risks.
- Lyra must be integrated into lore and UI.
- Lyra cannot directly mutate deterministic state.
- Lyra should preserve player agency.

## Checklist

- Is Lyra's advice useful but not overbearing?
- Does the response respect player knowledge and game state?
- Are Rune explanations actionable?
- Are risks and missing requirements clear?
- Is the UI role distinct from the narrative panel?

## Expected Output Format

- Lyra behavior summary.
- Inputs and context used.
- Suggested response style.
- Boundaries and risks.
- Tests/checks or review notes.
