# Image Generation Pipeline

Image generation is planned for post-MVP asset production, not as a dependency of the first gameplay loop.

## Planned Tools

- Stable Diffusion / SDXL.
- Stable Diffusion WebUI Forge.
- WAI Illustrious SDXL as a primary style candidate.
- Pony Diffusion V6 XL as a secondary style candidate.
- Future LoRA experiments for character and style consistency.

## Asset Categories

- Characters.
- Locations.
- Creatures.
- Maps.
- Narrative scenes.

## Rules

- Generated assets should be treated as production support, not core MVP infrastructure.
- Asset metadata should eventually link images to characters, locations, scenes, or events.
- Public repository language around visual content must remain professional and portfolio-safe.
