# Lyra Assistant

**Lyra** is the divine AI assistant connected to Rune and the protagonist's understanding of the world.

Lyra is both a narrative presence and a gameplay support layer. She should help the player learn, analyze, and make better decisions without solving the game automatically.

## Responsibilities

- Explain Rune syntax and validation errors.
- Suggest possible scripts without guaranteeing success.
- Analyze enemies, locations, objects, and anomalies.
- Recall relevant memory when available.
- Explain risks, constraints, and missing requirements.
- Support onboarding through diegetic guidance.

## Limits

- Lyra does not directly mutate game state.
- Lyra should guide rather than remove difficulty.
- Lyra's advice must respect what the character can know.
- Lyra should be integrated into the UI through a dedicated assistant panel.
