# Project Vision

**The World I Can Develop** is a local-first AI-driven fantasy text RPG. The internal codename is **Demon King**.

The project combines a structured RPG interface, deterministic game systems, persistent memory, local LLM narration, and a distinctive magic system called **Rune**. Rune lets the protagonist interact with the world through bounded divine pseudocode. **Lyra** is the divine AI assistant that helps the player understand Rune, analyze situations, and interpret risks without replacing player agency.

The project is designed as a professional portfolio repository. Its public presentation should emphasize architecture, local AI orchestration, deterministic state validation, memory design, and practical product thinking.

## Goals

- Build a text-first RPG that feels more structured than a chat window.
- Use local AI for narrative generation while preserving deterministic game rules.
- Create a memorable gameplay identity through Rune and Lyra.
- Develop in small, reviewable, documented phases.
- Keep the repository understandable for future collaborators and recruiters.

## What Makes It Strong

- Clear separation between generated narrative and validated state changes.
- Local-first design using Ollama and PostgreSQL.
- A distinctive pseudocode magic system that creates technical and gameplay depth.
- Documentation-first architecture suitable for a long-running project.
